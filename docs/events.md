# Events

Events are delivered to your inline handlers or `update()` function.
Each event has a `kind` discriminant field for type narrowing.

```typescript
import {
  isClick, isInput, isSubmit, isToggle, isSelect, isSlide,
  isKey, isMouse, isTimer, isAsync, isWindow, isEffect,
} from 'plushie'
```

## Widget events

Generated by user interaction with widgets. Widget events have
`kind: "widget"` and carry `id`, `scope`, `value`, and `data` fields.

### Click

```typescript
// Inline handler (preferred):
<Button id="save" onClick={handleSave}>Save</Button>

// Or in update():
if (isClick(event, 'save')) return save(state)
```

Generated by `Button` widgets when pressed.

### Input

```typescript
<TextInput id="search" value={state.query}
  onInput={(s, e) => ({ ...s, query: e.value as string })} />
```

Generated by `TextInput` on every keystroke. `event.value` is the
current text content.

### Submit

```typescript
<TextInput id="search" value={state.query}
  onSubmit={(s) => [s, searchCommand(s.query)]} />
```

Generated when the user presses Enter (when `onSubmit` is set).

### Toggle

```typescript
<Checkbox id="darkMode" value={state.darkMode}
  onToggle={(s, e) => ({ ...s, darkMode: e.value as boolean })} />
```

Generated by `Checkbox` and `Toggler`. `event.value` is boolean.

### Select

```typescript
<PickList id="size" options={['S', 'M', 'L']} selected={state.size}
  onSelect={(s, e) => ({ ...s, size: e.value as string })} />
```

Generated by `PickList`, `ComboBox`, and `Radio`.

### Slide

```typescript
<Slider id="volume" value={state.volume} range={[0, 100]}
  onSlide={(s, e) => ({ ...s, volume: e.value as number })}
  onSlideRelease={(s) => [s, Command.async(async () => saveVolume(s.volume), 'saved')]} />
```

`slide` fires continuously while dragging. `slide_release` fires
when the user releases the slider.

### Scoped events

Events from widgets inside named containers include a `scope` array
(reversed ancestor chain, immediate parent first):

```typescript
// Widget "toggle" inside container "todo_1" inside column "list":
// event.id === "toggle"
// event.scope === ["todo_1", "list"]
```

See [Scoped IDs](scoped-ids.md) for details.

## Keyboard events

Requires a subscription. Events have `kind: "key"`.

```typescript
subscriptions: (state) => [
  Subscription.onKeyPress('keys'),
],

update(state, event) {
  if (isKey(event, 'press')) {
    if (event.key === 's' && event.modifiers.ctrl) {
      return [state, saveCommand()]
    }
  }
  return state
}
```

Fields: `key`, `modifiedKey`, `physicalKey`, `modifiers`
(`{ ctrl, shift, alt, logo, command }`), `location`, `text`, `repeat`,
`captured`.

## Mouse events

Requires a subscription. Events have `kind: "mouse"`.

```typescript
subscriptions: (state) => [
  Subscription.onMouseMove('mouse', { maxRate: 30 }),
],
```

Types: `moved`, `entered`, `left`, `pressed`, `released`, `scrolled`.
Fields: `x`, `y`, `button`, `deltaX`, `deltaY`.

## Timer events

From `Subscription.every()`. Events have `kind: "timer"`.

```typescript
subscriptions: (state) => [
  Subscription.every(1000, 'tick'),
],

update(state, event) {
  if (isTimer(event, 'tick')) {
    return { ...state, time: new Date().toLocaleTimeString() }
  }
  return state
}
```

## Async events

From `Command.async()`. Events have `kind: "async"`.

```typescript
if (isAsync(event, 'fetchResult')) {
  if (event.result.ok) {
    return { ...state, data: event.result.value }
  } else {
    return { ...state, error: String(event.result.error) }
  }
}
```

## Stream events

From `Command.stream()`. Events have `kind: "stream"`. Each yielded
value produces a StreamEvent. The final return produces an AsyncEvent.

## Window events

Requires a subscription. Events have `kind: "window"`.

Types: `opened`, `closed`, `close_requested`, `resized`, `moved`,
`focused`, `unfocused`, `rescaled`, `file_hovered`, `file_dropped`.

## Effect events

From platform effects. Events have `kind: "effect"`.

```typescript
if (isEffect(event, 'openFile')) {
  if (event.status === 'ok') {
    return { ...state, filePath: event.result as string }
  }
  // status can be 'ok', 'cancelled', or 'error'
}
```

## Canvas events

From canvas widgets. Events have `kind: "canvas"`.

Types: `press`, `release`, `move`, `scroll`.
Fields: `id`, `x`, `y`, `button`.

Canvas shape events are widget events with types like
`canvas_shape_click`, `canvas_shape_enter`, `canvas_shape_leave`,
`canvas_shape_drag`.

## Mouse area events

From `MouseArea` widgets. Events have `kind: "mouse_area"`.

Types: `enter`, `exit`, `right_press`, `right_release`,
`middle_press`, `middle_release`, `double_click`, `move`, `scroll`.

## Modifiers events

From `Subscription.onModifiersChanged()`. Events have
`kind: "modifiers"`. Use `isModifiers(event)` to narrow.

## System events

Events have `kind: "system"`. Types include `animation_frame`,
`theme_changed`, `all_windows_closed`, `announce`.
